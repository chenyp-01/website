#### 2. 什么是SVG

svg是一种基于XML语法的图像格式，全称是可缩放矢量图（Scalable vector Graphics），其他图像格式都是基于像素处理的，svg是属于对图像的形状描述，所以他本质上是文本文件，体积较小，且不管放大多少倍都不会失真

svg文件可以直接插入页面，成为DOM的一部分，然后用javascript和css进行操作

#### 3. 显示svg图片的方式

1. 浏览器直接打开 （*.svg文件）

   资料中有tu.svg文件

2. 直接在 HTML 中使用 SVG 标签

```html
<svg  width="200" height="200">
	<!--脸-->        
	<circle cx="100" cy="100" r="90" fill="orange" />  
    
	<!--眼睛-->        
	<circle cx="70" cy="80" r="20" fill="white" />        
	<circle cx="130" cy="80" r="20" fill="white" />        
	<circle cx="65" cy="75" r="10" fill="black" />        
	<circle cx="125" cy="75" r="10" fill="black"/>  
    
	<!--嘴巴-->        
	<path d="M 50 140 A 60 60 0 0 0 150 140" 
          stroke="white" stroke-width="3" fill="none" />        
</svg>	
```

3. 在 HTML 中使用 <img> 标签引用

```html
<img src="images/tu.svg" alt="">
```

4. 作为 CSS 背景

   ```css
   .box {
       width: 500px;
   ```

5. 用js动态访问

   ```js
   //获取标签为svg的dom

   //获取svg中标签为circle的dom 
    var circle = document.querySelector("circle");
   
   svg.onclick = function () {
   	circle.style.display = "none";
   }
   ```

   

#### 4. svg预定义的形状元素

<rect>　矩形
<circle>　 圆形
<ellipse>　 椭圆
<line> 　线
<polyline> 　折线
<polygon> 　多边形
<path>　路径

基本属性：fill（填充）、stroke(描边)、stroke-width（描边宽度）、transform(变换)

```html
<svg>
         <!-- 矩形 -->
        <rect x="50" y="10" rx="5" ry="6" width="100" height="50" stroke="red" fill="none" transform="translate(80 0)"></rect>

        <!-- 圆形 -->
        <circle cx="100" cy="100" r="50" fill="green"></circle>

        <!-- 椭圆 -->
        <ellipse cx="100" cy="100" rx="50" ry="100" fill="yellow"></ellipse>

        <!-- 线 -->
        <line x1="0" y1="0" x2="200" y2="50" stroke="#f00" stroke-width="8"></line>

        <!-- 折线 -->
        <polyline points="0 0 50 50 200 50 100 200" stroke="#f00" stroke-width="10" fill="none"></polyline>

        <!-- 多边形 -->
        <polygon points="0 0 50 50 200 50 100 200" stroke="#f00" stroke-width="10" ></polygon>

        <!-- 自定义形状 -->
        <path d="M 50 50 L 100 100 200 30 100 250" stroke="red" fill="none"></path>

        <path d="M 50 50 L 100 100 Q 250 30 270 200" stroke="red" fill="none"></path>
    </svg>
```

- Path  路径
  <path>元素是SVG基本形状中最强大的一个，它不仅能创建其他基本形状，还能创建更多其他形状。
  path元素的形状是通过属性d定义的，属性d的值是一个“命令+参数”的序列

- 用于路径数据的命令 (仅作参考)
  M = moveto
  L = lineto
  H = horizontal lineto
  V = vertical lineto
  C = curveto
  S = smooth curveto
  Q = quadratic Belzier curve
  T = smooth quadratic Belzier curveto
  A = elliptical Arc
  Z = closepath

#### 5. svg的viewBox属性

viewBox就是截屏工具选中的那个框框，最终的呈现就是把框框中的截屏内容再次在svg中全屏显示！

viewBox="50,50,50,100"  : 前两个参数表示截屏起始点的坐标，后两个参数表示宽和高

```html
<svg viewBox="50,50,50,100">
    <!-- <svg> -->
        <g id="shape">
            <!--脸-->
            <circle cx="100" cy="100" r="90" fill="orange" />
            <!--眼睛-->
            <circle cx="70" cy="80" r="20" fill="white" />
            <circle cx="130" cy="80" r="20" fill="white" />
            <circle cx="65" cy="75" r="10" fill="black" />
            <circle cx="125" cy="75" r="10" fill="black" />
            <!--嘴巴-->
            <path d="M 50 140 A 60 60 0 0 0 150 140" stroke="white" stroke-width="3" fill="none" />
        </g>
    </svg>
```



#### 6. svg的元素（了解）

- <g>来实现图形群组。  

  要引用<g>元素，必须在<g>元素上设置一个ID，通过ID来引用它。<use>元素通过xlink:href属性来引入<g>元素。注意在ID前面要添加一个#。

- SVG的<defs>元素用于预定义一个元素使其能够在SVG图像中重复使用。

  在<defs>元素中定义的图形不会直接显示在SVG图像上。要显示它们需要使用<use>元素来引入它们。

```html
<svg>
<!-- 添加了defs时，原图形不再显示 -->
        <defs>
            <g id="shape">
                <!--脸-->
                <circle cx="100" cy="100" r="90" fill="orange" />
                <!--眼睛-->
                <circle cx="70" cy="80" r="20" fill="white" />
                <circle cx="130" cy="80" r="20" fill="white" />
                <circle cx="65" cy="75" r="10" fill="black" />
                <circle cx="125" cy="75" r="10" fill="black" />
                <!--嘴巴-->
                <path d="M 50 140 A 60 60 0 0 0 150 140" stroke="white" stroke-width="3" fill="none" />
            </g>
        </defs>
        <use xlink:href="#shape" x="100" y="100"></use>
        <use xlink:href="#shape" x="200" y="200"></use>
</svg> 
```

#### 7. svg 描边的特殊属性：
​	stroke-dasharray: 10,20,10,34     奇数项是虚线的长度，偶数项是间隙的宽度，可以设置多对数据
​	stroke-dashoffset： 表示虚线样式内虚线开始处的距离

  ```html

  </svg>
  ```

  ```css
      /* stroke-dasharray: 30 10; */
      /* stroke-dasharray: 100;
      stroke-dashoffset: 0;
      transition: all 2s; */
  }
  ```

#### 8. svg动画案例1

- html

  ```html
  <svg>
          <line x1="0" y1="150" x2="300" y2="150" stroke="#f00" stroke-width="3"></line>
          <path d="M 0 0 Q 100 20 100 150 Q 300 50 300 150 Q 250 250 0 300" stroke="#f00" stroke-width="3" fill="none"></path>
      </svg>
  ```

- css

  ```css
  svg {
      width: 300px;
      height: 300px;
      border: 1px solid #000;
      margin: 50px;
  }
  svg path {
      stroke-dasharray: 800;
      stroke-dashoffset: 800;
      transition: all 2s;
  }
  
  svg:hover path{
      stroke-dashoffset: 0;
  }
  ```

#### 9. svg动画案例2

沿路径运动的小球

  ```html

     <!-- <path d="M 0 0 Q 100 20 100 150 Q 300 50 300 150 Q 250 250 0 300" stroke="#f00" stroke-width="3" fill="none"></path> -->
  ```

- css

  ```css

      width: 300px;
      height: 300px;
      border: 1px solid #000;
      margin: 50px;
  }
  
  .ball {
      width: 50px;
      height: 50px;
      background-color: green;
      border-radius: 50%;

      offset-distance: 0%;
      animation: move 5s;
  }
  @keyframes move{
      0% {
          offset-distance: 0%;
      }
      100% {
          offset-distance: 100%;
      }
  }
  ```
