#### wx.getMenuButtonBoundingClientRect()
获取菜单按钮（右上角胶囊按钮）的布局位置信息。坐标信息以屏幕左上角为原点。获取值的单位为px，所以设置的时候也需要使用px进行设置<br />bottom: 56 屏幕顶部到按钮底部之间的距离<br />height: 32 按钮高度<br />left: 226 屏幕左边到按钮左侧之间的距离<br />right: 313 屏幕左边到按钮右侧之间的距离<br />top: 24 屏幕顶部到按钮顶部之间的距离<br />width: 87 胶囊按钮宽度<br />但是切换到不同尺寸的屏幕，获取到的菜单按钮的数据是不一样的，特别是top值，所以，当我们想要添加一个跟胶囊按钮同位置的input框的时候，可以通过getMenuButtonBoundingClientRect获取距离顶部的高度，再将值设置给input框<br />![image_XiyZm43fXt.png](https://cdn.nlark.com/yuque/0/2023/png/8438658/1692538571150-22d6390a-16bd-4064-85c6-b80e3911a296.png#averageHue=%23738ed7&clientId=u59afb8a0-aa90-4&from=paste&height=208&id=u9e2b9b58&originHeight=364&originWidth=664&originalType=binary&ratio=1.75&rotation=0&showTitle=false&size=307905&status=done&style=none&taskId=ue7039eae-b6b0-4c60-9088-64c89992f55&title=&width=379.42857142857144)<br />json配置

js
```
Page({
  /**
   * 页面的初始数据
   */
  data: {
    // 距离顶部的高度
    num1:0,
    // 胶囊的高度
    num2:0
  },
  onLoad(options) {
    let that = this;
    //获取胶囊信息
    let obj = wx.getMenuButtonBoundingClientRect();
    this.setData({
      num1:obj.top,
      num2:obj.height
    })
    console.log(obj);
  }
})
```
