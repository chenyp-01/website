# 1.发布小程序
- 在微信开发者工具中，单击右上角【上传】，提交开发版本

![image.png](https://cdn.nlark.com/yuque/0/2023/png/8438658/1692159766407-207c8db0-3f3d-4175-b556-5b370c05c7a3.png#averageHue=%23484747&clientId=ue4ae98fb-b359-4&from=paste&id=ue7e926b0&originHeight=101&originWidth=1918&originalType=binary&ratio=1&rotation=0&showTitle=false&size=28231&status=done&style=none&taskId=u3b99283f-0d6a-4846-8648-fb011d1446a&title=)

- 进入小程序管理后台 , 左侧菜单【管理】【版本管理】

![image.png](https://cdn.nlark.com/yuque/0/2023/png/8438658/1692159887503-6aca0f21-3c05-463c-a29d-12990e25c517.png#averageHue=%2373cc63&clientId=ue4ae98fb-b359-4&from=paste&height=921&id=u898fe1bd&originHeight=921&originWidth=1884&originalType=binary&ratio=1&rotation=0&showTitle=false&size=99941&status=done&style=none&taskId=u32d699a8-5be0-4b5d-bbbc-37376c277ab&title=&width=1884)

- 开发版本中【体验版】二维码，可供体验者扫码，之后会有消息发至管理员微信，管理员【通过】后，即可体验

- 单击开发版本右侧【提交审核】，根据提示填写【审核资料】，选填的都可以不填，其它都保持默认即可 

![image.png](https://cdn.nlark.com/yuque/0/2023/png/8438658/1692165211682-b7c085b2-8fb5-462b-9fa9-85f1007a4b2f.png#averageHue=%23fdfdfd&clientId=ue4ae98fb-b359-4&from=paste&height=919&id=ue58d75b9&originHeight=919&originWidth=1237&originalType=binary&ratio=1&rotation=0&showTitle=false&size=64822&status=done&style=none&taskId=uaccade39-2e9e-4efe-aadb-7cfdf6bf39a&title=&width=1237)

- 提交后，可看到审核版本，等待官方审核（2023年8月16日，审核时间非常快，可能不到10分钟就能审核通过）
- 审核通过后，即可发布
- 发布后即显示在线上版本

![image.png](https://cdn.nlark.com/yuque/0/2023/png/8438658/1692165268074-c128807b-3139-4601-8e17-d5dec4fe3339.png#averageHue=%23dce79d&clientId=ue4ae98fb-b359-4&from=paste&height=839&id=ub4af110c&originHeight=839&originWidth=1343&originalType=binary&ratio=1&rotation=0&showTitle=false&size=59698&status=done&style=none&taskId=u2f452a08-78ab-4f47-80ff-0ac397ac3bb&title=&width=1343)
# 2.生成小程序二维码

- 小程序管理后台，左侧最下方菜单 【设置】 【基本设置】

![image.png](https://cdn.nlark.com/yuque/0/2023/png/8438658/1692165331096-7c584457-1fe8-4efc-b87b-dcf078f9e2cc.png#averageHue=%2346bf45&clientId=ue4ae98fb-b359-4&from=paste&height=919&id=u0b1b47ed&originHeight=919&originWidth=1825&originalType=binary&ratio=1&rotation=0&showTitle=false&size=112311&status=done&style=none&taskId=u7b887c90-e629-4e8b-90eb-7d8a1553338&title=&width=1825)

- 基本信息中 【小程序码及线下物料下载】，单击最右侧【下载】，即可下载普通二维码和小程序码

![image.png](https://cdn.nlark.com/yuque/0/2023/png/8438658/1692165368938-3d2c9ea5-c282-4e0f-bdb7-0d5881ab8425.png#averageHue=%23fbfbfb&clientId=ue4ae98fb-b359-4&from=paste&height=611&id=u34ecd6fd&originHeight=611&originWidth=989&originalType=binary&ratio=1&rotation=0&showTitle=false&size=35519&status=done&style=none&taskId=u4d7a3b98-5441-49bf-b7be-31c75165af8&title=&width=989)
# 3.打开小程序

- 可在小程序中搜索小程序名称（基本信息中有小程序名称）
- 也可直接扫描二维码
