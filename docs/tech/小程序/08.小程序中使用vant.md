官网地址： [https://vant-contrib.oschina.io/vant-weapp/#/home](https://vant-contrib.oschina.io/vant-weapp/#/home)<br />使用步骤

-  打开终端,初始化，生成package.json 
> 项目目录不能有中文，必须是规范的nodejs项目目录

  

-  通过npm安装vant weapp 
```
npm i @vant/weapp -S --production
```


-  修改 app.json 
```
将 app.json 中的 "style": "v2" 去除，小程序的新版基础组件强行加上了许多样式，难以覆盖，不关闭将造成部分组件样式混乱。
```


-  修改 project.config.json 
```
{
  ...
  "setting": {
    ...
    "packNpmManually": true,
    "packNpmRelationList": [
      {
        "packageJsonPath": "./package.json",
        "miniprogramNpmDistDir": "./"
      }
    ]
  }
}
```


-  构建npm包<br />打开微信开发者工具，点击 **工具 -> 构建 npm**，构建完成后，即可引入组件。 
-  引入组件<br />以 Button 组件为例, 在app.json或page.json中 
```
"usingComponents": {
  "van-button": "@vant/weapp/button/index"
}
```


-  使用组件<br />引入组件后，可以在 wxml 中直接使用组件 
```
<van-button type="primary">按钮</van-button>
```
