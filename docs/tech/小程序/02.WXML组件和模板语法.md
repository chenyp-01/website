# 1.WXML中常用组件
1. 视图组件：  <view>
2. 文本组件：  <text>
3. 按钮组件： <button>
4. 图片组件： <image>
5. <block> 并不是一个组件，它仅仅是一个包装元素，不会在页面中做任何渲染，只接受控制属性。
6. 图标组件： <icon>
