# 1.配置app.json
在 app.json 中的 tabBar 项指定 custom 字段，同时其余 tabBar 相关配置也补充完整。
```json
"tabBar": {
    "custom": true,
    "color": "#000000",
    "selectedColor": "#000000",
    "backgroundColor": "#000000",
    "list": [
      {
        "pagePath": "pages/index/index",
        "text": "首页"
    	}, 
      {
        "pagePath": "pages/logs/logs",
        "text": "日志"
    	}
    ]
},
```
# 2.添加 tabBar 代码文件
在代码根目录下添加入口文件:
```
custom-tab-bar/index.js
custom-tab-bar/index.json
custom-tab-bar/index.wxml
custom-tab-bar/index.wxss
```
# 3. 编写 tabBar 代码
用自定义组件的方式编写即可，该自定义组件完全接管 tabBar 的渲染
## custom-tab-bar/index.js
```javascript
Component({
    data: {
      selected: 0,
      "color": "#000000",
        "selectedColor": "#000000",
        "backgroundColor": "#000000",
        "list": [{
            "pagePath": "/pages/index/index",
            "text": "首页"
        }, {
            "pagePath": "/pages/logs/logs",
            "text": "日志"
        }]
    },
    attached() {
    },
    methods: {
      switchTab(e) {
        const data = e.currentTarget.dataset
        const url = data.path
        wx.switchTab({url})
        this.setData({
          selected: data.index
        })
      }
    }
  })
```
## custom-tab-bar/index.json
```json
{
    "component": true,
    "usingComponents": {}
}
```
## custom-tab-bar/index.wxml
```html
<view class="tab-bar">
  <view class="tab-bar-border"></view>
  <view wx:for="{{list}}" wx:key="index" class="tab-bar-item" data-path="{{item.pagePath}}" data-index="{{index}}" bindtap="switchTab">
    <image src="{{selected === index ? item.selectedIconPath : item.iconPath}}"></image>
    <view style="color: {{selected === index ? selectedColor : color}}">{{item.text}}</view>
  </view>
</view>
```

# 4.自定义tabBar闪烁问题的解决

问题描述 ： 在切换进入其它tabBar页面时，第一个会先高亮，然后跳到第二个

原因： tabBar实质是在每个tabBar页面中单独引入的自定义组件，页面切换时，tabBar会重新渲染调用attach生命周期

解决方案： 用app.json中的全局变量来维护高亮的索引

案例代码：

app.js

```js
App({
    globalData: {
        tabIndex: 0
    }
})

```

page.js

```js
const app = getApp();
Component({
  data: {
    active:0,
    color: "#7A7E83",
    selectedColor: "#3cc51f",
    list: []
  },
 
  attached() {
    console.log('tabBar渲染');
    // 在组件实例进入页面节点树时执行, 从全局拉取最新的index
    this.setData({
      active:app.globalData.tabIndex
    })
  },
  
  methods: {
   
    //切换tabBar
    switchTab(e) {
      const {index,url} = e.currentTarget.dataset
      wx.switchTab({
        url
      })
      //把切换tabBar后最新的index存入全局
      app.globalData.tabIndex = index
    }
  }
})
```
