## 1. 配置css和js

1. 安装重置样式normalize.css

   ```
   npm i normalize.css
   ```

2. 移动端适配js

   复制flexible.js到src/assets文件夹中

3. 引入图标字体

   在assets/fonts中存入图标字体文件

   在assets/css中存入iconfont.css, 并修正路径

4. 定义全局css

   在assets/css中创建global.css,定义全局样式

5. 在main.js中引入以下文件 ，以便全局使用

   ```js
   // 引入重置样式
   import 'normalize.css'
   import './assets/css/global.css'
   import './assets/css/iconfont.css'
   // 引入rem适配的js
   import './assets/flexible'
   ```

6. 修改px-to-rem插件的配置

   基准font-size： 75

   规定屏幕宽度： 750

7. App.vue的style标签中，去掉scoped属性

8. 在任意组件中使用图标字体

   ```html
   <i class="iconfont icon-kefu"></i>
   ```

## 2. 处理本地静态资源

```vue
<div class="logo">
        <p><button>切换图片</button></p>
        <p><img :src="url" alt=""></p>
        <!-- <p><img src="../../../assets/img/m_hualogo.png" alt=""></p> -->
 </div>
```

```js
<script>
//引用本地静态资源模块
import logo from '../../../assets/img/m_hualogo.png'

export default {
    data() {
        return {
            //引用本地静态资源，不能使用下面的方式
            // url: '../../../assets/img/m_hualogo.png'
            url: logo,
            // 线上地址有效
            // url: 'https://m.hua.com/content/vue/login/static/img/m_hualogo.png'
        }
    }
}

</script>
```

## 3. vite 中使用 @ ，配置路径别名

修改vite.config.js

```js
import { defineConfig } from "vite";
import vue from "@vitejs/plugin-vue";
import path from 'path'
// https://vitejs.dev/config/
export default defineConfig({
  plugins: [vue()],
  server: {
    //启动热更新
    hmr: true,
  },
  resolve: {
    // 配置路径别名
    alias: {
      '@': path.resolve(__dirname, './src'),
    },
  },
});
```

## 4. Sass的使用

#### 1. 全局引入sass

- src/assets/scss/common.scss

  ```css
  $green: #0f0;
  ul,li {
      list-style: none;
      background-color: $green;
  }
  ```

- 在main.js中引入

  ```js
  import "@/assets/scss/common.scss"
  ```

#### 2. 在指定组件中引入sass

- src/assets/scss/variables.scss  （定义全量）

  ```css
  $bg: rgb(17, 24, 152);
  ```

- 在组件中引入

  ```css
  <style lang="scss">
      @import "./assets/scss/variables.scss";
      $light: #f00;
      .app {
        width: 5rem;
        border: 1px solid $light;
        background-color: $bg;
      }
  </style>
  ```
